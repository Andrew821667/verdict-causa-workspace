# Current.md - Verdict Causa

## Общая информация

- **Проект:** Verdict Causa
- **Версия институционального пакета:** `contracts-ru-v0@0.17.0`
- **Дата создания журнала:** 2026-07-23
- **Последнее обновление:** 2026-07-23

## Цель проекта

Создать проверяемую систему юридического анализа для российской правовой
системы. Система должна разделять доказанные факты, нормы, формальные выводы,
человеческую юридическую оценку и русское объяснение результата.

## Текущая стадия

- **Стадия:** разработка институционального слоя договорного права
- **Текущая задача:** временная передача незавершенного пилотного контура другому агенту
  после общих правил купли-продажи
- **Блокеры:** отсутствуют

## Технологический стек

- Python, Pydantic, Z3
- Pytest, Ruff
- JSON-артефакты с воспроизводимым replay
- GitHub Actions

## Архитектурные правила

- Каждый правовой блок имеет отдельные evidence, mapping, constraint model,
  benchmark, red-team и synthetic artifact.
- Неполные либо непроверенные факты отклоняются до формального решателя.
- Общие и специальные нормы исполняются последовательно и сверяются между собой.
- Формальный результат не подменяет судебную оценку.
- Юридически значимые документы и пользовательские объяснения создаются на русском.
